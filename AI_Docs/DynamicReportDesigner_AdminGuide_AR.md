# دليل إدارة التقارير الديناميكية

## إضافة تقرير

1. افتح شاشة إدارة التقارير من الـ Area الحالية.
2. اضغط `تقرير جديد`.
3. أدخل:
   - كود التقرير.
   - الاسم العربي/الإنجليزي.
   - نطاق المشروع.
   - نوع المصدر: Stored Procedure أو View.
   - اسم المصدر الآمن مثل `dbo.ReportName`.
   - MaxRows و Timeout.
4. اضغط `تحميل الأعمدة والباراميترز`.
5. راجع الأعمدة والمعايير.
6. اضغط `حفظ التقرير`.

## قواعد مهمة

- لا تضع SQL حر في SourceName.
- استخدم Stored Procedure أو View فقط.
- اجعل التقارير الثقيلة تتطلب تاريخ من/إلى داخل الـ SP.
- اضبط `MaxRows` بقيمة محافظة.
- المستخدم العادي لا يرى SourceName.

## الأعمدة

يمكن ضبط:

- Caption AR/EN.
- الظهور الافتراضي.
- قابلية الفلترة.
- قابلية الترتيب.
- قابلية التجميع.
- قابلية المجاميع.
- العرض والترتيب.

## الصلاحيات

التقرير يظهر للمستخدم إذا كان Admin أو له صلاحية في `DynamicReportPermissions`. التقارير Shared تظهر عبر المشاريع مع بقاء شرط الصلاحيات.
