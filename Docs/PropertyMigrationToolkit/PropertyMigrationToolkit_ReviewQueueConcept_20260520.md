# Property Migration Toolkit - Review Queue Concept
Date: 2026-05-20

## Screen Columns
- Priority.
- Severity.
- Warning Type.
- Entity Type.
- Source Record.
- Target Record.
- Original Value.
- Applied Fix.
- Fallback Entity.
- Suggested Action.
- Assigned To.
- Status.
- Resolution Notes.

## Actions
- Mark as Reviewed.
- Accept Fallback.
- Change Mapping.
- Exclude Record.
- Assign To User.
- Re-run Mapping for Record.
- Re-run Migration for Record.
- Export to Excel.

## Filters
- Critical only.
- Accounting only.
- Suspense only.
- Unknown units/properties.
- Payment methods.
- Owner payments.
- Unresolved items.

## GoLive Gate
The UI should clearly show whether open queue items block GoLive or only ReadyToTest.
