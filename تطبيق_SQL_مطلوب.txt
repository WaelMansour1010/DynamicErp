🔧 تطبيق Stored Procedure على قاعدة البيانات

الخطوات:

1. افتح SQL Server Management Studio

2. اتصل بقاعدة البيانات: MyErp

3. افتح الملف:
   📁 PropertyContract_Update_Fixed.sql
   (موجود في نفس المجلد)

4. نفذ الـ Script (F5 أو Execute)

5. تأكد من رسالة النجاح:
   "Command(s) completed successfully."

═══════════════════════════════════════════════════════════════

✅ بعد تطبيق الـ SQL Script، الميزات التالية ستعمل:

   ✓ تعديل الفرع بعد الحفظ
   ✓ تحديث السيريال تلقائياً
   ✓ تعديل الوحدة بعد الحفظ
   ✓ إرجاع الوحدة القديمة إلى "متاحة"
   ✓ تحديث الوحدة الجديدة إلى "مشغولة"
   ✓ التعامل الذكي مع الوحدات المدمجة
